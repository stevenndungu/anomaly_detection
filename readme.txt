Galaxy recognition from galaxy source images with trainable COSFIRE filters

To run the experiment described in [1]
    1. Change to the folder called Application   
    2. Run "setup;"
    2. Read the help contents of the script called Galaxy_recognition_with_COSFIRE.m
    3. Run "result = Gender_recognition_with_COSFIRE(100,'/scratch/p307791/radio/',150,5.00,0:5:20,0.05,0.50,0.15,1,10,1000);"
